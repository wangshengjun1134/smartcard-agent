# SmartcardAgent 设计文档

## 项目概述

SmartcardAgent 是一个智能卡知识库代理桌面应用，帮助用户管理和查询智能卡相关的知识和数据。采用 Tauri + React + Python 三层架构设计。

## 系统架构

### 三层架构

```
┌─────────────────────────────────────────────────────────┐
│                    Tauri 桌面应用层                      │
│  (Rust 后端 - 系统级调用、跨平台集成、窗口管理)          │
├─────────────────────────────────────────────────────────┤
│                    React 前端层                          │
│  (TypeScript/React - 用户界面、交互逻辑、状态管理)       │
├─────────────────────────────────────────────────────────┤
│                Python 知识库服务层                       │
│  (独立进程 - 知识存储、检索、智能卡数据处理)             │
└─────────────────────────────────────────────────────────┘
```

### 技术栈

| 层级 | 技术 | 版本 | 说明 |
|------|------|------|------|
| 桌面应用 | Tauri | 2.x | 跨平台桌面应用框架 |
| 前端框架 | React | 18 | 用户界面框架 |
| 前端语言 | TypeScript | 5.x | 类型安全的 JavaScript |
| 构建工具 | Vite | 5.x | 前端构建和开发服务器 |
| 后端服务 | Python | 3.11+ | 知识库处理服务 |
| 包管理 | uv/pip | - | Python 包管理器 |

### 知识库页面

**Header**: 仅右侧保留 `+` 按钮，用于添加新知识

**内容区域**:
- 搜索栏：带有搜索图标，占位符 "搜索知识库..."
- 知识卡片网格：`grid-cols-1 md:grid-cols-2 lg:grid-cols-3`
- 卡片内容：图标、标题、描述、文档数量、更新时间

### 技能库页面

**Header**: 仅右侧保留 `+` 按钮，用于创建新技能

**内容区域**:
- 搜索栏：带有搜索图标，占位符 "搜索技能..."
- 技能卡片网格：同知识库布局
- 卡片内容：图标、标题、描述、标签、使用次数

## 核心组件

### Sidebar 侧边栏

**结构**:
```
┌─────────────────────────────────┐
│  [新建对话按钮]                  │
│  [知识库导航] [技能库导航]        │
│  ─────────────────────          │
│  [分组列表]                      │
│    [会话项...]                   │
│  ─────────────────────          │
│  [用户信息区域 (Buff)]           │
│    [用户菜单弹出]                │
└─────────────────────────────────┘
```

**功能**:
- 新建对话按钮
- 视图切换导航（聊天/知识库/技能库）
- 分组管理（展开/折叠、重命名、删除）
- 会话管理（切换、重命名、删除、固定）
- 用户菜单触发区域

### UserMenu 用户菜单

**触发**: 点击侧边栏底部 Buff 区域

**菜单项**:
- 设置（打开 SettingsDialog）
- 深色模式/浅色模式（切换主题）
- 客服中心
- 退出登录

### SettingsDialog 设置对话框

**尺寸**: `780px × 560px`

**左侧菜单**:
- 账号
- 通用
- 知识库
- 技能库
- 关于

**通用设置内容**:
- 字号大小（滑块：更小/标准/更大/超大）
- 主题色彩（下拉选择：浅色模式）
- 启动设置
- 默认设置
- 文件下载位置
- 下载前询问（开关）
- 下载完成后显示（开关）

### ChatHeader 聊天顶部栏

**左侧**: 折叠按钮（侧边栏关闭时显示） + 模型选择下拉
**右侧**: API服务、历史、画笔、帮助按钮组

## 主题系统

### ThemeContext

**状态管理**:
- `isDarkMode`: boolean - 当前是否为深色模式
- `toggleDarkMode(): void` - 切换主题

**持久化**:
- localStorage 存储主题偏好
- 支持 `prefers-color-scheme: dark` 系统偏好检测

**CSS 实现**:
- `.dark` 类添加到 `document.documentElement`
- Tailwind `dark:` 前缀处理组件样式
- `globals.css` 定义全局深色模式覆盖规则
